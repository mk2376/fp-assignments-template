# fp-assignments

[VSCode esxtension](https://open-vsx.org/vscode/item?itemName=azdavis.millet)

Enter:

```sh
devbox shell
```

Run commands:

```sh
task -l # List all

# Examples
task run FILE=00/00.sml
task interpret FILE=00/00.sml
task test FILE=00/00.sml
```
